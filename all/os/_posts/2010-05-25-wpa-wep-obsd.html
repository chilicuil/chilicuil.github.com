WPA - ip estática

A partir de la versión 4.4 (4.5 para ath0) se ha integrado soporte para wpa, no todos los drivers lo soportan, pero para los que si lo hacen, un index rápido de los comandos son (aquí para atheros): 

    $ ifconfig ath0 nwid ACCESS_POINT \
      wpa wpapsk $(wpa-psk ACCESS_POINT   PASSWORD)

    $ ifconfig ath0 10.0.0.2 255.255.255.0 10.0.0.1

También se puede agregar a /etc/hostname.ath0, para que se conecte al arranque: 

    $ cat /etc/hostname.ath0

    inet 10.0.0.2 255.255.255.0 10.0.0.255 \
    nwid ACCESS_POINT wpa wpapsk \
    0xc7bd82ef64a789369e18d6df63230a3b099f72a74b999bdbe837773e6081cb54

Donde el último parámetro se obtiene con $ wpa-psk ACCESS_POINT PASSWORD

WPA - ip dinámica

    $ ifconfig ath0 nwid ACCESS_POINT \
      wpa wpapsk $(wpa-psk ACCESS_POINT PASSWORD)

    $ dhclient ath0

Y agregandolo a /etc/hostname.ath0 quedaría: 

    $ cat /etc/hostname.ath0

    dhcp nwid ACCESS_POINT wpa wpapsk \
    0xc7bd82ef64a789369e18d6df63230a3b099f72a74b999bdbe837773e6081cb54

WEP - ip estática

    $ ifconfig ath0 nwid ACCESS_POINT nwkey 0xPASSWORD
    $ ifconfig ath0 10.0.0.2 255.255.255.0 10.0.0.1

Y agregandolo a /etc/hostname.ath0 quedaría: 

    $ cat /etc/hostname.ath0
    
    inet 10.0.0.2 255.255.255.0 10.0.0.255 \
    nwid ACCESS_POINT nwkey 0xPASSWORD

WEP - ip dinámica

    $ ifconfig ath0 nwid ACCESS_POINT nwkey 0xPASSWORD
    $ dhclient ath0

Y agregandolo a /etc/hostname.ath0 quedaría: 

    $ cat /etc/hostname.ath0
    dhcp nwid ACESS_POINT nwkey  0xPASSWORD
